## 项目简介
**Jianhong_labelimg** 是一款基于经典开源工具 `labelImg` 深度定制的增强版标注软件。它不仅保留了原有工具的简洁性，更针对工业级大规模标注场景，引入了**自动化工作流统计**、**序列帧智能复制**以及**防崩溃保护**等企业级功能。

为什么选择 Jianhong_labelimg？
- **效率**: 批量复制功能可将标注速度提升 3-5 倍。
- **可靠**: 完善的撤回机制和防闪退逻辑，保障标注数据安全。
- **透明**: 实时统计系统，让每日进度一目了然。

## 核心特性

### 📊 智能工作流统计
- **每日零点重置**: 自动根据系统时间，在每日凌晨 12:00 清空计数。
- **分组明细**: 按图片存储路径自动分组显示，方便多任务管理。
- **实时看板**: 在主界面顶部直观展示今日已标注总量。

### ⚡ 序列帧效率工具
- **智能覆盖复制**: 选定标注框后，支持 `Alt+C` 批量复制到后续 N 帧。
- **数据纯净性**: 复制过程中会自动清空目标帧的原有标注，避免重复和冲突。
- **多步状态恢复**: `Ctrl+Z` 支持多达 10 步的历史记录恢复。

### 🛡️ 工业级稳定性
- **IO 异常捕捉**: 针对网络驱动器或权限受限目录，提供友好的错误提示而非直接崩溃。
- **状态机优化**: 彻底修复了在绘制过程中触发快捷键导致的逻辑混乱问题。
- **单文件分发**: 支持 PyInstaller 打包为纯 EXE，无需配置环境即可运行。

## 快速开始

### 安装方式
1. 克隆仓库：
   ```bash
   git clone https://github.com/2859363572/Jianhong_labelimg.git
   cd Jianhong_labelimg
   ```
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 启动程序：
   ```bash
   python run.py
   ```

## 快捷键参考

| 动作 | 默认快捷键 | 功能描述 |
| :--- | :--- | :--- |
| **创建选框** | `W` | 进入交互式画框模式 |
| **撤回操作** | `Ctrl + Z` | 回退上一次操作（支持10步） |
| **批量复制** | `Alt + C` | 唤起自定义复制对话框 |
| **下一张图** | `D` | 保存并加载下一张 |
| **删除选框** | `Del` / `F` | 移除当前选中的标注对象 |
| **快捷键配置** | `Ctrl + H` | 打开自定义快捷键界面 |

## 常见问题 (FAQ)
**Q: 如何修改批量复制的默认帧数？**  
A: 按下 `Alt+C` 后，在弹出的对话框中输入目标帧数即可。系统会自动记录您的上次输入。

**Q: 软件在保存时报错 "Permission Denied" 怎么办？**  
A: 请确保目标目录具有写入权限，且 XML/TXT 文件未被其他程序占用。本项目已加入保护逻辑，报错时会通过状态栏提示而非闪退。
