## 项目简介
**Jianhong_labelimg** 是基于经典的 `labelImg` 进行二次开发的增强版图像标注工具。它旨在通过 AI 驱动的自动化逻辑和深度优化的用户体验，解决大规模数据标注中的效率和稳定性痛点。

相比原版，本项目新增了**实时统计、批量复制、多步撤回**等核心功能，特别适合视频帧标注和大规模目标检测数据集的快速制作。

## 核心特性

### 📊 智能工作流
- **每日图片统计**: 自动按日期、按路径分组统计已完成图片数量，每日零点自动清空。
- **实时状态显示**: 主界面顶部实时显示图片目录、保存目录及今日工作总量。

### ⚡ 效率倍增器
- **自定义帧数复制**: 选中标注框后，支持一键批量复制到后续 N 帧，并自动清除目标帧原有冗余数据。
- **多步撤回机制**: 支持高达 10 步的操作撤回（Ctrl+Z），覆盖创建、删除、移动、修改等所有核心操作。
- **自定义快捷键**: 提供独立的快捷键配置界面，允许用户根据操作习惯自由定义。

### 🛡️ 系统稳定性
- **异常保护机制**: 深度优化了保存逻辑，遇到路径权限、文件占用等问题时通过状态栏友好提示，防止软件闪退。
- **状态机优化**: 解决了画框过程中触发快捷键导致的逻辑冲突问题。

## 快速开始

### 环境要求
- Windows / macOS / Linux
- Python 3.9+
- PyQt5

### 安装方式
1. 克隆仓库：
   ```bash
   git clone https://github.com/2859363572/Jianhong_labelimg.git
   cd Jianhong_labelimg
   ```
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 启动程序：
   ```bash
   python run.py
   ```

## 快捷键指南

| 动作 | 默认快捷键 | 说明 |
| :--- | :--- | :--- |
| **创建选框** | `W` | 进入画框模式 |
| **撤回操作** | `Ctrl + Z` | 恢复到上一步状态（支持10步） |
| **自定义复制** | `Alt + C` | 批量复制选中框到后续 N 帧 |
| **下一张图** | `D` | 保存并切换到下一张 |
| **上一张图** | `A` | 切换到上一张 |
| **删除选中** | `Del` / `F` | 删除当前选中的标注框 |

## 技术架构
- **核心框架**: PyQt5
- **数据存储**: Settings 机制持久化配置与统计数据
- **文件处理**: 增强型 `LabelFile` 类，支持 PascalVOC (XML), YOLO (TXT)
- **打包方案**: PyInstaller One-File 模式

## 常见问题 (FAQ)
**Q: 为什么我的统计数据每天都会消失？**  
A: 这是设计使然。系统会根据本地时间在每日凌晨 12:00 清空计数，以便您开始新一天的统计。

**Q: 批量复制时会覆盖已有的框吗？**  
A: 是的。为了保证标注序列的一致性，批量复制会清空目标帧原有的标注，仅保留当前帧复制过去的框。
